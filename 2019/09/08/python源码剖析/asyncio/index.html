异步原理

loop.run_until_complete(test())

将test()封装为Future，或者说Task，存到消息队列里面去，下一次循环执行这个test(),由于是一个类生成器函数，需要Task来控制这个生成器的恢复执行开始执行，一个Future执行完后（set_result），会添加它之前设置好的callback到Loop里面。这个回调通常用来恢复test()协程函数。
